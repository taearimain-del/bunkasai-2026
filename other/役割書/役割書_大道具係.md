# 大道具係
## 役割書 — 大道具係

---

## 概要・責任範囲

大道具係は**セット（背景・大型の舞台装置）の制作・設置・場面転換**と**教室装飾**を担当する。
公演当日は暗転中に素早くセットを入れ替える必要があるため、転換フローを全員で把握しておくこと。

---

## 必要人数・役割分担

| 人数 | 役割 |
|------|------|
| 1〜2人 | 制作リーダー（設計・材料調達） |
| 2〜3人 | 制作・塗装作業 |
| 全員（本番） | 場面転換担当（暗転中に素早く動く） |

**目安：3〜4人（本番の場面転換には出番のない役者も加わる）**

---

## 制作物リスト

### セット

| 品目 | 使用シーン | サイズ感 | 制作方法（案） |
|------|-----------|---------|-------------|
| 背景パネル：氷・雪（青白） | ACT3『レット・イット・ゴー』＆ACT5 エルサの城 | 黒板サイズ程度 | 青白い布または大きな画用紙。雪の結晶を貼る |
| 背景パネル：城内（茶系） | ACT2 戴冠式・ACT6 室内 | 黒板サイズ程度 | 段ボールや布で城の壁をイメージして制作 |
| ドアフレーム | ACT4 納屋（扉）のみ | 人が通れる大きさ | 木材または段ボールで枠だけ作る（扉自体はなくてよい）。ACT4→5の暗転で撤去後は再使用なし |

> ※ 実際の大きさ・仕様はリハーサル室の広さを確認してから決定する。

### 教室装飾（当日）

| 場所 | 品目 | 制作方法（案） |
|------|------|-------------|
| 教室入口 | 演目名の看板 | 大きな紙に手書き or 印刷 |
| 教室入口〜廊下 | 雪の結晶デコレーション | 折り紙・画用紙で制作。青・白・水色で統一 |
| 廊下 | 開演時刻・次回の開始時刻の案内 | A4で印刷して掲示 |
| 舞台周辺 | キャスト紹介ボード | 役名と役者名を書いたボード |

## 本番の場面転換フロー

暗転中に速やかに動く。演出または舞台監督の合図（手信号）で動き出すこと。

| タイミング | 転換内容 | 担当 |
|-----------|---------|------|
| ACT1→ACT2（雪だるまつくろう後） | 戴冠式用の背景・小道具を配置 | 大道具係＋役者（ハンス役・公爵役など） |
| ACT2→ACT3（とびら開けて後） | 不要なセットを退ける | 大道具係 |
| ACT3『レット・イット・ゴー』**開始時** | 白・水色の布/パネルを後方に展開（雪山・氷の城を表現） ※暗転ではなく歌の始まりに合わせて出す | 大道具係 |
| ACT3→ACT4（レット・イット・ゴー後） | 白・水色パネルを撤去＋ドアフレームをACT4（納屋の扉）の位置に設置 | 大道具係＋役者（トロール役） |
| ACT4→ACT5（あこがれの夏後） | 氷の城背景パネル（白・水色）を前面に設置＋ACT4の納屋ドアフレームを撤去 | 大道具係 |
| ACT5→ACT6（ハンスの部屋へ） | 氷の城パネル（白・水色）を撤去・茶系背景パネルに切り替え。机・椅子は小道具係がセット | 大道具係 |
| ACT6内（クライマックス前・短い暗転） | 机・椅子を舞台袖へ素早く撤収。茶系背景パネルはそのまま残す（城の外壁として転用） | 大道具係 |

## 準備スケジュール（目安）

| 時期 | 作業内容 |
|------|---------|
| 7月前半 | 制作物リストを確定・材料の調達計画を立てる |
| 7月後半〜8月前半 | 背景パネル・ドアフレームの制作 |
| 8月後半 | セットを実際に舞台に置いて通し稽古に合わせる |
| 9月前半 | 教室装飾の制作 |
| 9月7〜9日 | 会場入り・セット配置確認・転換の通し練習 |
| 本番前日 | 教室装飾を完成させる |

## 役者との連携

出番の少ない役者が場面転換を手伝う。事前に動きを確認しておくこと。

| 役者 | 手伝えるシーン |
|------|-------------|
| ハンス役 | ACT1→2 転換（自分の登場シーンの直前） |
| ウェーゼルトン役 | ACT3以降の転換 |
| トロール役 | ACT1〜4の転換（登場前） |

---

## リハーサルで確認すること

- [ ] 各セットが暗転中に1分以内で転換できるか確認
- [ ] ドアフレームが倒れないよう補強しておく
- [ ] 背景パネルの固定方法（立て掛け・壁への取り付け等）を決めておく
- [ ] 転換中に音を立てないよう練習する
