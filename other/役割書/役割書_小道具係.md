# 小道具係
## 役割書 — 小道具係

---

## 概要・責任範囲

小道具係は**本番に使うすべての小道具の管理・出し入れ**を担当する。
役者に渡し忘れると本番で演技が止まるため、チェックリストで毎回確認すること。

---

## 必要人数・役割分担

| 人数 | 役割 |
|------|------|
| 1人 | 小道具リーダー（チェックリスト管理） |
| 1〜2人 | 各シーンの出し入れ担当 |

**目安：2〜3人**

---

## 小道具リスト

| 品目 | 数量 | 使用シーン | 調達方法 |
|------|------|-----------|---------|
| 杖（王権の杖） | 1本 | ACT2 戴冠式（エルサが持つ） | 手作りまたは購入 |
| 玉（王権の玉） | 1個 | ACT2 戴冠式（エルサが持つ） | 手作りまたは購入 |
| エルサの白手袋 | 1双 | ACT2 戴冠式（手袋をはめる演技） | 手芸店 |
| 人参 | 1本 | ACT4 オラフの鼻に刺す | 本物 or フェイク（安全のため模造品推奨） |
| ロープ | 1本 | ACT4 クリストフへ渡す | ホームセンター |
| 金づち | 1本 | ACT4 クリストフへ渡す | 家から持参 or ホームセンター |
| ハンスの剣（模造） | 1本 | ACT6 クライマックス | 玩具コーナー or 手作り |
| 暖炉プロップ（段ボール製・LEDライト内蔵） | 1個 | ACT6 ハンスの部屋 | 段ボール・スプレー塗料・電池式LEDライトで手作り |
| 書類（書き物） | 数枚 | ACT6 ハンスの机の上 | 普通の紙でOK |
| 地図風の紙 | 数枚 | ACT6 ハンスの机の上 | 紙に地図風の絵を描いて用意 |
| 羽ペン風小道具 | 1本 | ACT6 ハンスの机の上 | 100均等で調達 or 手作り |

---

## 小道具の出し入れキュー

| タイミング | 対応内容 |
|-----------|---------|
| ACT2 開幕前（暗転中） | 杖・玉を大人エルサ役の登場位置にセット。白手袋は開演30分前に大人エルサ役へ直接手渡し済みであることを確認する（台本line83「渡す」指示に準拠） |
| 戴冠式終わり（手袋着用後） | 担当者が素早く杖・玉を舞台裏に回収 |
| 戴冠式終わり（照明が消えた直後） | 大人エルサの白手袋を回収。大人エルサが袖へはける際に手渡しを受けるか、袖口で受け取る |
| ACT3終わり（レット・イット・ゴー後の暗転中） | 大人エルサが脱いだマントを小道具係が素早く回収。舞台に放置しないこと |
| ACT4 開幕前（暗転中） | ロープ・金づち・人参を夏アナ役者の手の届く位置に準備 |
| ACT4：人参を渡すシーン | 夏アナ役者が人参をオラフの顔に刺した後、スヴェン役が狙う小道具として機能 |
| ACT6 開幕前（暗転中） | ①机の上に書類・地図風の紙・羽ペン風小道具を並べる（机・椅子の搬入は大道具係）。②暖炉プロップを舞台端の壁際に設置し、LEDライトを点灯する。**照明点灯の5秒前に点灯しておくこと**（点灯済みであることを照明係と舞台監督に確認）。オラフがスイッチを入れやすい向きに調整する。③ハンスの剣は預かっておく（着席して書き物をするACT6冒頭では渡さない）。配置完了後、ハンスが着席して照明点灯を待つ |
| ACT6 クライマックス前の暗転 | 舞台袖でハンス役へ剣を手渡す（上手から剣を持って登場するのに間に合わせる） |
| ACT6：ハンスが剣を引き抜くシーン | クライマックス前の暗転で剣を渡し済みか確認しておく |
| 本番終了後 | 全小道具を回収し、リストと照合してチェック |

---

## 当日の確認手順

本番開始前に必ず全員でチェックすること：

- [ ] 杖・玉がACT2の配置位置にある
- [ ] 白手袋が開演30分前に大人エルサ役へ直接手渡し済みであることを確認した
- [ ] 人参・ロープ・金づちがACT4の配置位置にある
- [ ] 暖炉プロップのLEDライトが点灯確認済み
- [ ] ハンスの剣を小道具係が管理している（ACT6クライマックス前の暗転でハンス役へ手渡す段取りを確認）
- [ ] 書類・地図風の紙・羽ペン風小道具がACT6用に準備されている

---

## オラフ人参管理フロー

ACT4でオラフの鼻に装着した人参は、ACT6終了まで**鼻につけたまま継続**する（スヴェンの演技との視覚的一貫性のため）。

| タイミング | 対応 |
|-----------|------|
| ACT3 line306 | 小道具係がオラフ役者の鼻に人参を装着 |
| ACT3→ACT4暗転 | 人参はつけたまま袖へ（取り外し不要） |
| ACT4→ACT5暗転 | 人参はつけたまま維持 |
| ACT5→ACT6暗転 | 人参はつけたまま継続 |
| ACT6終了後 | 小道具係が回収・保管 |

> ⚠️ 役者が各自で取り外さないよう事前に伝えておくこと。

---

## 役者との連携

出番の少ない役者が小道具の受け渡しを手伝うことがある。

| 役者 | 手伝えること |
|------|------------|
| ハンス役（ACT1・4・5中） | 舞台袖での小道具の準備補助 |
| オラフ役（ACT3中） | 小道具の確認作業補助 |

---

## リハーサルで確認すること

- [ ] 各小道具の受け渡しタイミングをリハーサルで確認する
- [ ] 暗転中に素早く設置できる「配置場所」を固定する
- [ ] 暖炉プロップのスイッチ操作（消し方・つけ方）をハンス役・オラフ役と事前に練習する
- [ ] 予備品が必要かどうかを確認する（人参は消耗品なので複数用意すること）
