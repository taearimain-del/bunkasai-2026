舞台監督

役割書 — 舞台監督

概要・責任範囲

舞台監督は公演全体の進行を管理する司令塔。

照明・音響・大道具・小道具・役者のすべてに指示を出す。

  • 「誰が何をいつやるか」を全員に伝える役
  • 本番中は声が出せないため、手信号でコントロールする
  • 練習から本番まで、計画どおりに進んでいるかを常に確認・修正する

必要人数・役割分担

人数役割
1人 舞台監督本体（本番中の全指示を担当）
1人（できれば） 副監督（タイム計測・メモ・舞台監督のサポート）

目安：1〜2人

合図の方法

セリフ中は声を出せないため、手信号で各係に合図を出す。

事前に全係と統一しておき、リハーサルで必ず練習すること。

合図意味
手を上げる 「準備してください」
手を下ろす 「今すぐ操作してください（実行）」
人差し指を立てる 「1分前」
両手を広げる 「止まれ・ちょっと待って」
親指を立てる 「OK・次へ進んでいい」
手を2回叩く 「暗転を延長します（+15秒）」

場面転換の4ステップ（毎回この順番で）

暗転が入るたびに、必ずこの順序で確認・指示する：

1. 役者が舞台袖に引っ込んだか確認

2. 大道具・小道具に転換の合図を出す

3. 各係から「完了」のサインが来たら照明係に点灯の合図

4. 役者に「入れ」の合図を出す

⚠️ 手順3で全係の完了を確認しないまま次に進まないこと。
※ 歌唱中に同時進行する場面（レット・イット・ゴー）は別フロー参照（下記「歌唱中同時操作シーン対応」セクション）。

リハーサル管理

舞台監督はリハーサルで「本番どおりに動けているか」を確認し、問題があれば次回までに修正する。

リハーサルごとのチェック項目

チェック内容確認方法
暗転の時間が 30秒〜1分以内に収まっているか タイム計測（副監督が計る）
小道具の受け渡しミスがないか 各シーンを見て確認
夏アナの旅用衣装への早替えがACT3→4の暗転（30秒〜1分）内に間に合っているか 夏アナ役者の着替え完了を確認
照明・音響のキューのタイミングがズレていないか 台本のキューと照らし合わせる
役者のセリフや立ち位置に問題がないか 通し稽古中にメモする
場面転換の動線が渋滞していないか 舞台袖と舞台の動きを観察
ACT5→6転換（45秒）が完了できているか ゲネプロで必ず計測・秒数を記録する
延長合図（手を2回叩く）が全係に伝わっているか 使用シーンを全員で事前確認

リハーサル後にやること

1. 問題点をメモにまとめ、関係する係に伝える

2. 「次の練習までに直す」こと・「本番まで練習が必要なこと」を区別してリスト化

3. 全体的なタイム（合計上演時間）を記録する

練習スケジュールの目安

時期練習内容
7月〜8月前半 パート別練習（役者は台本読み・各係は準備・制作）
8月後半 ACTごとの部分通し（照明・音響のタイミングを合わせる）
9月前半 全体通し稽古（衣装・小道具も含めた本番と同じ形式）
9月10日（目安） ゲネプロ（本番前日の最終リハーサル）。時間を計測する
本番前日 会場のセットアップ・全小道具の配置確認

ゲネプロ（最終リハーサル）でチェックすること

  • [ ] 全体の上演時間が 35〜45 分に収まっているか
  • [ ] すべての暗転が 1分以内に完了しているか
  • [ ] ACT5→6転換（45秒）の完了時間を計測・記録した（机椅子搬入・小道具セット・エルサ動線すべて含む）
  • [ ] すべての係が台本のキューを見ずに動けているか
  • [ ] トラブルが起きたときの対応（代役・カット案）を確認した

歌唱中同時操作シーン対応

暗転のない歌唱中に大道具・照明が同時に動く唯一のシーン（レット・イット・ゴー）の合図フロー。

レット・イット・ゴー 歌唱中合図フロー（暗転なし）

```

1. 大人エルサが歌い出しの1フレーズ前（前奏終了）でスタンバイ

2. 歌い出しの瞬間：照明係へ「照明GO」合図（舞台内ライト点滅1回で通知）

3. 同時：大道具係へ「パネル展開GO」合図（舞台内ライト点滅1回）

4. 大道具係は上手2名が黒スタッフ服で、8小節を目安にパネルを展開

```

マント脱ぎ照明合図（キュー#11b）：

  • 大人エルサが両腕を広げる（マントを脱ぐ直前）のタイミングで照明係へ「最大輝度GO」合図（照明卓隣に立ち肩タップ）
  • 担当：舞台監督または副監督

ACT5→6 暗転（45秒）のタイムライン

```

暗転開始

0〜 5秒：大人エルサが上手袖から舞台裏へ進入（上手袖担当が通過確認）

0〜20秒：大道具4名が上手・下手両袖から氷パネルを撤去→茶系パネルに切替

5〜15秒：大人エルサが舞台裏を横断し下手袖へ到達・待機

20〜35秒：大道具2名が上手袖から机・椅子を搬入（大人エルサ通過確認後）

20〜35秒：小道具2名が下手袖から書類・地図・羽ペンをセット。暖炉プロップを舞台端の壁際に設置・LED点灯

※ 下手袖：大人エルサは上手寄り1.5m、小道具は下手寄りで並行作業

35〜40秒：大人エルサが下手袖から舞台中央へ移動・震えのポーズ

40〜45秒：全員スタンバイ完了・点灯待ち

```

合図の順序：

1. 暗転と同時に「上手袖担当→大人エルサ進入」

2. 上手袖担当が「エルサ通過確認」を肉声で舞台監督に報告

3. 報告を受けてから「机・椅子搬出GO」合図を出す

4. 35秒時点でパネル切替未完了の場合は即「延長合図（手を2回叩く）」→最大+15秒

ACT6内クライマックス前暗転（45秒）のタイムライン

```

暗転開始

0〜20秒：大道具が机・椅子を上手袖へ撤収（大人エルサ通過後）

大人エルサ：上手袖→舞台裏横断→下手袖待機

ハンス：上手袖で剣を受け取りスタンバイ

冬アナ・クリストフ・スヴェン・オラフ：下手袖へ移動

35〜40秒：大人エルサが下手袖から舞台中央へ移動

40〜45秒：全員スタンバイ完了

```

ACT4→ACT5暗転（大人エルサ入場方向）

  • 暗転中に大人エルサへ「上手から入る」合図を出す
  • アナ一行の下手スタンバイを確認してから、両方へ「GO」合図を出す順序で実施

トラブル対応

トラブル対応
役者がセリフを忘れた 舞台袖から小声でヒントを出す担当を事前に決めておく
照明の操作ミス 次のキューで自然に修正。本番中は急がない
音響の曲が出ない 音響係に両手広げの「止まれ」合図 → 待つ
夏アナの旅用衣装着替えが暗転内に間に合わない 暗転を少し延ばす（照明係に「待て」の合図を出し、夏アナ役者の着替え完了を確認してから点灯）
小道具を置き忘れた 即興で役者がセリフを変えるか省略する（事前に「○○がないときはこうする」を決めておく）
時間が大幅に押している 事前に「削れるセリフ・シーン」を決めておき、舞台監督の合図で実行する
ACT5→6転換が45秒を超えそう 35秒時点でパネル未完了なら即「延長合図（手を2回叩く）」。最大+15秒（計60秒）上限
