## 重要ルール

1. **削除・上書き前は必ず `過去版/` へアーカイブ**
   - 対象：台本・企画書・その他すべての編集・削除対象ファイル
   - 命名規則：`元ファイル名_YYYYMMDD.拡張子`（同日2回目以降は `_20260616v2` のように連番）
   - `git mv` でなく `cp` でコピーしてから編集すること（元ファイルはそのまま残す）
   ```powershell
   # 例：企画書.pdf を上書きする前に
   cp 生徒用\企画書.pdf 過去版\企画書_20260616.pdf
   git add 過去版\企画書_20260616.pdf
   ```

2. **台本の内容変更は `other/台本/台本_アナと雪の女王.md` を編集する**

3. **GitHubに置くもの**: PDF（生徒用/）・HTML＋.md（other/）・旧版（過去版/）のみ

4. **作業後は必ず `生徒用/変更点.md` を更新する**
   - 形式：`## YYYY-MM-DD HH:MM　タイトル`
   - 何をどう変えたかを箇条書きで記載（細かい時刻まで含める）
   - 最新エントリをファイル先頭（既存エントリより上）に追加する

5. **大事なことはこのファイル（プロジェクト概要.md）に追記する**
   - ユーザーから新しいルール・方針・決定事項を伝えられたとき
   - 作業中に「これは次回も守るべき」と判断したとき
   - 追記したら忘れずにコミット・プッシュする

6. **演出・小道具は学校の環境で実現可能なものにする**
   - 暖炉・本物の炎などは使えない → 今回はランタン（電池式）に置き換え済み
   - 台本を編集する際、「これは学校の舞台で本当にできるか？」を必ず確認する
   - 不可能な演出が見つかったら代替案を考えて台本・企画書を同時に更新する

7. **役割書（`other/役割書_〇〇係.md`）は台本と同じ扱いで管理する**
   - 変更前に現行PDFを `過去版/` にアーカイブ
   - 変更後はHTML→PDF再生成（上記「PDF再生成手順」参照）
   - 役割書は照明・音響・大道具・小道具・舞台監督の5種が存在する（衣装係は2026-06-28廃止）

8. **変更点.md を書く前に必ず `Get-Date -Format "yyyy-MM-dd HH:mm"` で現在時刻を取得する**
   - 日付はシステムから渡されるが時刻は自動では分からない。必ずコマンドで取得すること
