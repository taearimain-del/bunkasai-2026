# 台本 問題点まとめ

作成日：2026年6月16日

## ✅ 問題①【解決済み】歌唱7曲で約19分 → 上演時間を約40分に延長して解決

| 曲名 | 尺 |
|------|-----|
| 雪だるまつくろう | 約3分 |
| 生まれてはじめて | 約2分半 |
| とびら開けて | 約2分 |
| レット・イット・ゴー | 約4分 |
| あこがれの夏 | 約2分 |
| 生まれてはじめて（リプライズ） | 約2分半 |
| 愛さえあれば | 約3分 |
| **合計** | **約19分** |

30分公演で歌が19分 ＋ 場面転換（6幕×約30秒＝約3分）＝ セリフに使える時間は**約8〜9分**。
現状の台本のセリフ量はその2倍以上ある。

**対策案（どちらかまたは両方）:**
- 曲を1〜2曲カットする（候補：「あこがれの夏」「愛さえあれば」）
- 各幕のセリフを大幅に削る

## ✅ 問題②【解決済み】ACT6のナレーターが情報詰め込みすぎ

現状：
> 「ハンスのもとへたどり着いたアナは、エルサとの出来事、真実の愛があれば助かるということを話しました。」

この1文の直後にハンスが急に裏切りを告白するため、流れが唐突。

**対策案:**
- ハンスとアナの短い会話（1〜2往復）を挟む
- またはナレーターを削り、ハンスが部屋に入ってくるところから始める

---

## ✅ 問題③【解決済み】ACT5でアナが魔法に当たる場面のト書きが抜けている

エルサの城のシーン。いつアナが撃たれたのかト書きがなく、次のシーンで突然「髪が白くなってないか？」となる。観客に伝わらない。

**対策案:**
- 「生まれてはじめて（リプライズ）」の歌唱中または直後に、エルサの魔法がアナに当たるト書きを追加する

---

## ✅ 問題④【解決済み】スヴェンの演技指示がほぼない

キャスト表に「四つん派いで演じる」とあるが、台本本文中の指示は「人参に反応して飛びかかろうとする」の1箇所のみ。クライマックスでも登場するのに何のト書きもない。

**対策案:**
- 各場面でスヴェンの動きをト書きに明記する（例：クライマックスで駆けつけるシーン）

---

## ✅ 問題⑤【解決済み】フィナーレのオラフの締め台詞が弱い

現状：オラフ「なんか…これが愛ってやつ？」

テーマを回収する重要な場面なのにセリフが1行で終わっている。

**対策案:**
- もう1〜2行セリフを足してオラフらしい締めにする

---

## ✅ 問題⑥【解決済み】ハンス退場の演出が地味

現状：「そう言い放ち、顔面を殴った。ハンスは退場。」のみ。

**対策案:**
- ハンスの情けない反応をト書きに書く（例：尻もちをついて退場 など）
- または一言セリフを加えて爽快感を出す
